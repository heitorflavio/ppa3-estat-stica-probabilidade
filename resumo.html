    Cálculo de Média, Mediana e Moda
      Como Calcular a Média

      A média é a soma de todos os valores dividida pela quantidade de valores. A fórmula para calcular a média é: 

      
Média = (Soma de todos os valores) / (Quantidade de valores)

      Exemplo com as Alturas: Se tivermos as alturas 1.75, 1.80, 1.85, a média será: 

      
Média = (1.75 + 1.80 + 1.85) / 3 = 1.80
      Como Calcular a Mediana

      A mediana é o valor que se encontra no meio de um conjunto de dados ordenados. Se o número de dados for ímpar, a mediana é o valor central. Se o número for par, é a média dos dois valores centrais. A fórmula para calcular a mediana é: 

      
Mediana = valor no meio dos dados ordenados

      Exemplo com as Alturas (dados ordenados): 1.60, 1.70, 1.80. Como há 3 valores (número ímpar), a mediana é o valor central, que é 1.70.

      Como Calcular a Moda

      A moda é o valor que mais se repete em um conjunto de dados. Se houver mais de um valor com a mesma frequência máxima, o conjunto de dados terá mais de uma moda. A fórmula para calcular a moda é: 

      
Moda = valor com maior frequência

      Exemplo com as Alturas: Se tivermos as alturas 1.75, 1.75, 1.80, a moda é 1.75, pois é o valor que mais se repete.

      Como Calcular as Frequências

      Passo a Passo para Calcular as Frequências

      Para calcular as frequências de um conjunto de dados, siga os seguintes passos:

      1. Organizar os dados: Ordene os dados de forma crescente. Por exemplo, se você tiver alturas de alunos, coloque-as em ordem do menor para o maior.
      2. Definir os intervalos: Divida os dados em intervalos de acordo com a faixa de valores. Por exemplo, para as alturas, você pode usar intervalos como 1.60 - 1.64, 1.65 - 1.69, etc.
      3. Contar as ocorrências: Para cada intervalo, conte quantos valores caem dentro dele. Esse número é a frequência do intervalo. Por exemplo, se houver 5 alunos com altura entre 1.70 e 1.74, a frequência desse intervalo é 5.
      4. Calcular a Frequência Relativa: A frequência relativa (\( p_i \)) de um intervalo é a razão entre a frequência do intervalo e o número total de observações. Use a fórmula: 
        p_i = f_i / N
        Onde \( f_i \) é a frequência do intervalo e \( N \) é o número total de dados.
      5. Calcular a Frequência Acumulada: A frequência acumulada (\( F_i \)) é a soma das frequências dos intervalos até o intervalo \( i \). Use a fórmula: 
        F_i = Σ f_j
        Onde \( f_j \) é a frequência de cada intervalo até o intervalo \( i \).
      Exemplo Prático

      Suponha que você tenha os seguintes dados de altura (em metros) de 10 alunos: 

      
          1.60, 1.65, 1.70, 1.72, 1.75, 1.75, 1.80, 1.80, 1.85, 1.90

      Para calcular as frequências, você pode seguir esses passos:

      1. Organize os dados: 1.60, 1.65, 1.70, 1.72, 1.75, 1.75, 1.80, 1.80, 1.85, 1.90
      2. Defina os intervalos: Por exemplo: 1.60 - 1.64, 1.65 - 1.69, 1.70 - 1.74, 1.75 - 1.79, 1.80 - 1.84, 1.85 - 1.90.
      3. Conte as frequências:
        • 1.60 - 1.64: 1 aluno
        • 1.65 - 1.69: 1 aluno
        • 1.70 - 1.74: 2 alunos
        • 1.75 - 1.79: 2 alunos
        • 1.80 - 1.84: 2 alunos
        • 1.85 - 1.90: 2 alunos
      4. Calcule as frequências relativas (exemplo para o intervalo 1.60 - 1.64): 
        p_1 = f_1 / N = 1 / 10 = 0.1
      5. Calcule as frequências acumuladas:
        • F_1: A frequência acumulada do primeiro intervalo (1.60 - 1.64) é igual à sua frequência, ou seja: 
          F_1 = 1
        • F_2: A frequência acumulada do segundo intervalo (1.65 - 1.69) é a soma das frequências do primeiro e segundo intervalos: 
          F_2 = 1 + 1 = 2
        • F_3: A frequência acumulada do terceiro intervalo (1.70 - 1.74) é a soma das frequências dos três primeiros intervalos: 
          F_3 = 1 + 1 + 2 = 4
        • F_4: A frequência acumulada do quarto intervalo (1.75 - 1.79) é a soma das frequências dos quatro primeiros intervalos: 
          F_4 = 1 + 1 + 2 + 2 = 6
        • F_5: A frequência acumulada do quinto intervalo (1.80 - 1.84) é a soma das frequências dos cinco primeiros intervalos: 
          F_5 = 1 + 1 + 2 + 2 + 2 = 8
        • F_6: A frequência acumulada do sexto intervalo (1.85 - 1.90) é a soma das frequências dos seis primeiros intervalos: 
          F_6 = 1 + 1 + 2 + 2 + 2 + 2 = 10
